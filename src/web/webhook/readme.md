
### Setup

Example build with getting pings when Github issues created

1. Run
    ```
    python app.py
    ```
2. Install [ngrok](https://ngrok.com/) and expose port with 
    ```
    ngrok http 5000
    ```
3. Go to any Github Repo, Settings -> Webhooks
